Referência: https://blog.miguelgrinberg.com/post/the-flask-mega-tutorial-part-i-hello-world

1. Criar virtual environment
 virtualenv venv

2. Instalar o flask
 pip install flask

2.1 Gerar o arquivo com as dependências
 pip freeze > requirements.txt

3. Executar a aplicação
3.1 Definir a variável de ambiente:
 set FLASK_APP=microblog.py
 3.2 Executar o comando
  flask run

Observação: para realizar o auto carregamento da aplicação ao alterar arquivos deve-se adicionar os seguintes comandos:
 set FLASK_DEBUG=1
 python -m flask run

 4. Instalar dependências para base de dados
 pip install flask-sqlalchemy
 pip install flask-migrate

 5. Criar estrutura da base de dados
 flask db init
  5.1 Criar a migração mais recente
  flask db migrate -m "users table"
  5.2 Executar a migração
  flask db upgrade

 6. Instalar flask-login
